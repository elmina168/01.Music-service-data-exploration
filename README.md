# Music-service-data-exploration
Исследование данных музыкального сервиса - География успеха: что делает трек виральным в разных странах?
# Анализ глобальных трендов Spotify

## Описание проекта
Проанализировала 200 записей треков из глобального чарта Spotify. Выявила ключевые географические закономерности виральности треков, определила жанровые предпочтения в разных странах и построила корреляционные модели успеха.

## Цели
- Очистить и подготовить данные Spotify (потоки, виральность, жанры, страны)
- Провести географический анализ виральности треков
- Выявить жанровые предпочтения в разных странах
- Построить корреляционные модели между метриками успеха

## Используемые технологии
- Python 
- Pandas, NumPy — обработка данных
- Matplotlib, Seaborn — визуализация
- SciPy — корреляционный анализ

## Ключевые результаты 

### География успеха
- **Мексика и Колумбия**: 100% треков имеют тренд `Rising` (быстрорастущие рынки)
- **Корея**: 100% треков имеют тренд `Falling` (рынок быстрого потребления)
- **США**: только 33% треков растущие, 67% падающие

### Жанровое лидерство по странам
- Пуэрто-Рико / Колумбия → **Reggaeton** (viral_score ~33 млн)
- Мексика → **Regional Mexicano** (viral_score ~14 млн)
- США → **Alternative Pop** (viral_score ~42 млн)
- Великобритания → **Rock / New Wave** (viral_score ~18-25 млн)

### Корреляции виральности
- **США**: сильная связь с потоками (r = 0.71)
- **Корея**: отрицательная связь с изменением потоков (r = -0.44)
- **Мексика**: слабая связь с потоками, сильная с позицией в чарте
